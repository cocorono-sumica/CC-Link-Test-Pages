JavaScriptを使用した即座のリダイレクトです。以下のようなパターンが検出されます:

// パターン1: location.href window.location.href = "url"; location.href = "url"; // パターン2: location.replace window.location.replace("url"); // パターン3: location代入 window.location = "url"; location = "url";

このテストページでは、実際のリダイレクトは無効化されています。 拡張機能はJavaScriptを実行せず、HTMLソースコードの解析のみでリダイレクトを検出します。
