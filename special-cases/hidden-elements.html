検証ポイント

  • 各種CSSで非表示にされた要素が正しく検出される
  • 非表示理由が詳細に表示される
  • 「非表示」バッジが表示される
  • 「非表示エラー」フィルターで表示できる
  • 非表示の理由リストが表示される

テスト結果の確認方法

  1. 拡張機能のアイコンをクリックして起動
  2. 「Check Links」ボタンをクリック
  3. 結果を確認:
    • すべてのリンクに 👁️‍🗨️ 非表示 バッジが表示される
    • 非表示理由が詳細に表示される（display:none、visibility:hidden など）
    • 404エラーも同時に検出される
  4. フィルターで「非表示エラー」を選択し、非表示要素のみ表示できることを確認

💡 非表示要素の検出について

  • 拡張機能は以下の方法で非表示を検出します:
    • display: none
    • visibility: hidden
    • opacity: 0
    • サイズが0の要素（width: 0, height: 0）
    • 画面外に配置された要素
  • 非表示要素のエラーは、実際のユーザーには見えないため、優先度が低い場合があります
  • しかし、コード品質や将来の保守性のために修正を推奨します
